# Designs Folder

This folder is for designer-provided UI components and assets.

## Structure

```
designs/
├── components/     # React components from designer
├── pages/          # Page layouts from designer
├── assets/
│   ├── images/     # Image assets
│   ├── icons/      # Icon assets
│   └── fonts/      # Font files
└── styles/         # Global styles/CSS
```

## Usage

When you receive designs from your designer, place them in the appropriate subfolder. The integration guide will help connect them to the existing backend.
